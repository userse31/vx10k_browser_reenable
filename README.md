Reenables the web browser on the LG Voyager (vx10000). As an added bonus, it messes around with the AT modem interpreter inside the phone to allow for (janky and quite slow) dynamic memory dumps and patching.

The details of the AT modem thing can be read here: https://www.tumblr.com/userse31/811007681987051520/live-ram-dumps-on-the-lg-voyager?source=share

This program has no GUI or anything and immediately exits when done. Perhaps not the **best** idea to not give any form of visual/auditory feedback that it did anything, but hey, I'm lazy!
